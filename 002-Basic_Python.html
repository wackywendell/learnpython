Variables¶

A variable is a name we give to a piece of data. Variables are the building blocks of code.

In [2]:
x = 2
y = 3
my_string = "abcdef" # A "string" is a piece of text

You can print out the value of a variable

In [3]:
x = 10
print(x)

10

Variables can be reassigned.

In [4]:
x = 2
print(x)
x = 3
print(x)

2
3

Note that the equals sign (=) is an assignment operator: it says "I want to call the thing on the right by the name on the left"

Plotting with Lists¶

In Python, the plot function takes two lists as arguments: a list of x-coordinates, and a list of y-coordinates. That is the bare minimum you need to plot.

In [15]:
xs = [0, 1, 2, 3]
ys = [0, 2, 1, 3]
plot(xs, ys)
Out[15]:
[<matplotlib.lines.Line2D at 0x7f7e1c1b43c8>]

You can plot multiple data sets on the same plot.

In [16]:
xs = [0, 1, 2, 3]
ys = [0, 2, 1, 3]
plot(xs, ys)

ys2 = [0.1, 1, 1.5, 0.5]
plot(xs, ys2)
Out[16]:
[<matplotlib.lines.Line2D at 0x7f7e1c153f60>]

Arrays¶

An array is a like a list, but for just numbers. It allows you to do math on the whole array.

In [17]:

ys = array([0, 2, 1, 3])
plot(xs, ys)

ys2 = ys - xs
plot(xs, ys2)

[<matplotlib.lines.Line2D at 0x7f7e1c15d940>]
